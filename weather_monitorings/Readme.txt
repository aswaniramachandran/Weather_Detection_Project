Weather Monitoring Application


This project is a Weather Monitoring Application that fetches real-time weather data for multiple cities and alerts users when temperatures exceed a threshold.
The app dynamically updates weather information, displays detailed daily summaries, and allows users to switch between Celsius and Fahrenheit temperature units

Technologies Used

Frontend: HTML, CSS, JavaScript
API: OpenWeather API
Framework/Library: Django
Database: sqlite3(default db)

Prerequisites
Before you begin, make sure you have the following installed:

A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
Text editor (VS Code, Sublime, etc.)
API Key from OpenWeather (free signup required)


Installation

pip install django

Clone the repository:

git clone https://github.com/aswaniramachandran/Projects.git

Navigate to the project directory:

cd weather-monitoring-app


Set up the OpenWeather API key:

Open the weather.js file.
Replace the placeholder API_KEY with your actual OpenWeather API key
const API_KEY = 'your_openweather_api_key';

Open the project in a web browser:

Simply open the index.html file in any modern browser by double-clicking it or right-clicking and selecting "Open With."

How to Run the Application

python manage.py runserver
